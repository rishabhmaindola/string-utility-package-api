String Utility API Documentation

This documentation provides an overview of the available API endpoints for the String Utility service.

Available Endpoints

Endpoint Method Parameters Description
/api/v1/reverse GET text: string
(Required) The string to be reversed. 		Returns the reversed version of the input string.

Example Request

GET /api/v1/reverse?text=hello

Response:

"olleh"
/api/v1/isPalindrome GET text: string
(Required) The string to be checked. 		Returns true if the input string is a palindrome, otherwise false.

Example Request

GET /api/v1/isPalindrome?text=radar

Response:

true
/api/v1/duplicateCharacters GET text: string
(Required) The string in which duplicate characters are to be found. 		Returns a map of duplicate characters and their counts.

Example Request

GET /api/v1/duplicateCharacters?text=hello

Response:

{"l": 2}
/api/v1/checkRotation GET text1: string
text2: string
(Both required) Two strings to check if they are rotations of each other. 		Returns true if one string is a rotation of the other, otherwise false.

Example Request

GET /api/v1/checkRotation?text1=waterbottle&text2=bottlewater

Response:

true
/api/v1/toUpperCase GET text: string
(Required) The string to convert to uppercase. 		Returns the input string converted to uppercase.

Example Request

GET /api/v1/toUpperCase?text=hello

Response:

"HELLO"
/api/v1/toLowerCase GET text: string
(Required) The string to convert to lowercase. 		Returns the input string converted to lowercase.

Example Request

GET /api/v1/toLowerCase?text=HELLO

Response:

"hello"
/api/v1/randomString GET length: int
(Required) The length of the random string to generate. 		Generates a random alphanumeric string of the specified length.

Example Request

GET /api/v1/randomString?length=10

Response:

"Ab12cD3EfG"
/api/v1/isAnagram GET text1: string
text2: string
(Both required) Two strings to check if they are anagrams of each other. 		Returns true if the two strings are anagrams, otherwise false.

Example Request

GET /api/v1/isAnagram?text1=listen&text2=silent

Response:

true
/api/v1/toAlphaNumeric GET text: string
(Required) The string to convert to alphanumeric format. 		Returns a string containing only alphanumeric characters.

Example Request

GET /api/v1/toAlphaNumeric?text=Hello!@#$%^&*()_+123

Response:

"Hello123"
/api/v1/substringOccurrences GET text: string
substring: string
(Both required) The string to search in and the substring to search for. 		Returns the number of occurrences of the substring in the input string.

Example Request

GET /api/v1/substringOccurrences?text=hellohello&substring=lo

Response:

1
/api/v1/capitalizeFirst GET text: string
(Required) The string to capitalize the first letter of. 		Returns the input string with the first letter capitalized.

Example Request

GET /api/v1/capitalizeFirst?text=hello

Response:

"Hello"
/api/v1/stringLength GET text: string
(Required) The string to get the length of. 		Returns the length of the input string.

Example Request

GET /api/v1/stringLength?text=hello

Response:

5
/api/v1/camelToSnake GET text: string
(Required) The camelCase string to convert. 		Returns the input string converted to snake_case.

Example Request

GET /api/v1/camelToSnake?text=camelCaseString

Response:

"camel_case_string"
/api/v1/snakeToCamel GET text: string
(Required) The snake_case string to convert. 		Returns the input string converted to camelCase.

Example Request

GET /api/v1/snakeToCamel?text=snake_case_string

Response:

"snakeCaseString"
/api/v1/slugGenerator GET text: string
(Required) The string to generate a slug from. 		Returns a URL-friendly slug from the input string.

Example Request

GET /api/v1/slugGenerator?text=Hello World!

Response:

"hello-world"
/api/v1/isValidEmail GET email: string
(Required) The email to validate. 		Returns true if the input is a valid email format, otherwise false.

Example Request

GET /api/v1/isValidEmail?email=test@example.com

Response:

true
/api/v1/isValidURL GET url: string
(Required) The URL to validate. 		Returns true if the input is a valid URL format, otherwise false.

Example Request

GET /api/v1/isValidURL?url=https://example.com

Response:

true
/api/v1/isStrongPassword GET password: string
(Required) The password to validate. 		Returns true if the password is strong, otherwise false.

Example Request

GET /api/v1/isStrongPassword?password=StrongP@ssw0rd

Response:

true
/api/v1/emojiRemoval GET text: string
(Required) The string from which to remove emojis. 		Returns the input string with emojis removed.

Example Request

GET /api/v1/emojiRemoval?text=Hello👋😊

Response:

"Hello"
/api/v1/hammingDistance GET str1: string
str2: string
(Both required) Two strings to compare. 		Returns the Hamming distance between the two strings.

Example Request

GET /api/v1/hammingDistance?str1=karolin&str2=100101

Response:

3
/api/v1/base64Decode GET encoded: string
(Required) The Base64-encoded string to decode. 		Returns the decoded string from the input Base64-encoded string.

Example Request

GET /api/v1/base64Decode?encoded=SGVsbG8gd29ybGQ=

Response:

"Hello world"
/api/v1/base64Encode GET text: string
(Required) The string to Base64-encode. 		Returns the Base64-encoded string from the input string.

Example Request

GET /api/v1/base64Encode?text=Hello world

Response:

"SGVsbG8gd29ybGQ="
